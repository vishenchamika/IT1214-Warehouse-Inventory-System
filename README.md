# IT1214-Warehouse-Inventory-System
## 📌 Project Description
This is a simple Java-based Warehouse Inventory System that helps manage products, stock levels, and inventory operations.

## ⚙️ How to Run the Program
1. Open terminal / command prompt
2. Compile:
   javac Main.java
3. Run:
   java Main

## 📊 Sample Output
Enter product name: Apple  
Enter quantity: 10  
Product added successfully!
